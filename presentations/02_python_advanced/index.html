Гибкие настройки: *args и **kwargs

Используем упаковку аргументов для гибкого конфигуратора сайта:

def configure_site(site_name, *flags, **settings):
    print(f"Сайт: {site_name}")
    print(f"Активные режимы (args): {flags}") # Кортеж
    print(f"Параметры сайта (kwargs): {settings}") # Словарь

# Вызов с разными наборами данных:
configure_site("Квант-2026", "DEBUG_MODE", "MAINTENANCE", 
               port=8000, theme="light", max_users=150)
# args: ('DEBUG_MODE', 'MAINTENANCE')
# kwargs: {'port': 8000, 'theme': 'light', 'max_users': 150}

Суть распаковки

*args собирает позиционные аргументы в tuple.
**kwargs собирает именованные аргументы в dict.

Моделирование пользователей: User и Teacher

Используем защищенные атрибуты (_email) и декоратор @property для валидации:

class User:
    def __init__(self, username, email, role="student"):
        self.username = username
        self._email = email # Защищенное поле
        self.role = role

    @property
    def email(self):
        return self._email.lower()

    @email.setter
    def email(self, new_email):
        if "@" not in new_email:
            raise ValueError("Некорректный адрес почты")
        self._email = new_email

Сеттер перехватывает попытку изменения: user.email = "new@kvant.ru" и валидирует её.

Наследование: Класс Teacher

Наследуем Teacher от User, расширяя его полями учебного предмета:

class Teacher(User):
    def __init__(self, username, email, subject):
        # Вызов конструктора базового класса User
        super().__init__(username, email, role="teacher")
        self.subject = subject # Специализированное поле преподавателя

    # Полиморфизм: переопределяем строковое представление
    def __str__(self):
        return f"Преподаватель {self.username} (Предмет: {self.subject})"

teacher = Teacher("alex_phys", "alex@kvant.ru", "Физика")
print(teacher) # Выведет: Преподаватель alex_phys (Предмет: Физика)

Мероприятия школы: Класс Event

Класс Event использует dunder-методы для вывода в логи и списки:

class Event:
    def __init__(self, name, speaker, start_time):
        self.name = name
        self.speaker = speaker # Объект User или Teacher
        self.start_time = start_time # datetime

    def __str__(self):
        # Понятно пользователям (print)
        return f"Событие '{self.name}' (Спикер: {self.speaker.username})"

    def __repr__(self):
        # Детально разработчикам (отладка)
        return f"Event(name='{self.name}', speaker={repr(self.speaker)})"

Функции высшего порядка и Замыкания

В Python функции — это объекты первого класса:

  • Их можно записывать в переменные.
  • Передавать как аргументы в другие функции.
  • Возвращать из функций.

Замыкание (closure) — функция, которая помнит переменные из своего внешнего контекста (области видимости), даже когда внешняя функция завершила работу.

def make_notifier(prefix):
    # Внутренняя функция-замыкание
    def notify(message):
        print(f"[{prefix}] {message}")
    return notify

info_notify = make_notifier("КВАНТ-ИНФО")
info_notify("Обед через 10 минут!")
# Вывод: [КВАНТ-ИНФО] Обед через 10 минут!

Декоратор ролей: @requires_role

Декоратор с параметром оборачивает представление и проверяет роль пользователя:

def requires_role(required_role):
    def decorator(func):
        def wrapper(user, *args, **kwargs):
            if user.role != required_role:
                return f"Ошибка 403: Доступ запрещен для {user.username} (нужна роль {required_role})"
            return func(user, *args, **kwargs)
        return wrapper
    return decorator

@requires_role("teacher")
def create_event(user, event_name):
    return f"Событие '{event_name}' успешно создано преподавателем {user.username}"

Время проведения событий (datetime)

Для корректного расписания используем модуль datetime и временную зону UTC:

  • Парсинг строки времени: strptime.
  • Форматирование в строку: strftime.
  • Операции со временем возвращают объект timedelta.
from datetime import datetime, timezone, timedelta

# Текущее время в UTC
now = datetime.now(timezone.utc)

# Парсинг времени лекции
lecture_time = datetime.strptime(
    "2026-06-26 10:00:00", 
    "%Y-%m-%d %H:%M:%S"
).replace(tzinfo=timezone.utc)

# Разница во времени
time_left = lecture_time - now
print(f"До лекции осталось: {time_left}")

Регулярные выражения (Модуль re)

Regex (регулярные выражения) — шаблоны для поиска и проверки строк.

Спецсимволы:

  • ^ / $ — начало / конец строки.
  • \d — цифра, \w — буква, цифра или знак _.
  • {m,n} — от m до n повторений.
  • + — 1 или более, * — 0 или более.

Валидация регистрации на сайте:

import re

# Логин: от 3 до 15 латинских букв или цифр
login_pattern = r"^[a-zA-Z0-9_]{3,15}$"

def validate_username(username):
    if re.match(login_pattern, username):
        return True
    return False

print(validate_username("ivan_99")) # True
print(validate_username("a!")) # False

Архитектура ядра сайта школы Квант

  3. События: Создаются объекты Event с привязкой спикера и времени datetime.
  4. Доступ: Защита методов создания событий с помощью декоратора @requires_role.
